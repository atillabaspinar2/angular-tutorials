
# Dependency Injection

Definition of DI. Singleton.
Use injectable in a service
and either inject() or constructor. this is the most common way.

## Using provider in root

If the service is not decorated with injectable
1 - it can be registered to root app provider

## Using provider in components

2 - it can be used in a component's provider. but in this case, each instance of a component
will create a different instance of the service. all childrent of the component can reach to the service

providers can only be used in components, not in other services, pipes, directives.
